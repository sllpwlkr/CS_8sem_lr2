### Инструкция по установке и запуску
1. Установить ollama с сайта https://ollama.com/
2. Загрузить модель qwen в bash:
   ```bash
   ollama run qwen2.5:0.5b
   ```
3. Убедиться, что сервер ollama работает
4. Установить Python-зависимости:
   ```python
   pip install requests
   ```
5. Запустить скрипт:
   ```bash
   python ollama_qwen.py
   ```
